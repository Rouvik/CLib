◆ CLib_Vector_insert()

void CLib_Vector_insert ( CLib_Vector_t * vector,
int position,
TYPE data )

Inserts and sets (if the element is already present) to the CLib_Vector.

Note
Behaviour Description:
  • If position is less than size: Sets the element at the position with the data
  • If position is greater than or equal to size: Adds 0 values to the uninitialised elements and sets the element at position
    All of the above operations will resize the vector if required
Parameters
vectorThe pointer to the CLib_Vector to insert into
positionThe position to insert into
dataThe data to insert at position
