◆ CLib_String_substrView()

CLib_String CLib_String_substrView ( CLib_String * str,
unsigned int start,
unsigned int length )

Returns a CLib_String view of the substring part of the provided CLIB_String str this view shares the same memory as *str and *may or may not be null terminated must be used in tandem with the .len member of CLib_String, to mark it as a view, the .buf_len parameter of CLib_String is set to 0
Note: The returned CLib_String (view) must not be freed with CLib_String_deinit() calls, as it shares memory with *str

Parameters
strThe CLib_String to extract the substring from
startThe start location of the substring
lengthThe length of the substring
Returns
CLib_String The CLib_String(view, ie. .buf_len = 0, read above) containing the pointer to appropriate string and its length
